# 🧥 Trench Coat Shop Manager

A sleek C++ desktop application built with the Qt Widgets framework to manage a virtual trench coat boutique. This project features a dual-mode graphical user interface (GUI), providing separate tools and functionalities for shop administrators and regular customers. 🛍️✨

## Features

### 👨‍💼 Admin Mode
Designed for backend inventory management, the admin mode includes full CRUD capabilities:
* **Create:** Add new trench coat styles to the shop's inventory.
* **Read:** View the complete list of available coats and their specific details (price, size, color, stock, and image link).
* **Update:** Modify existing trench coat details or update stock quantities.
* **Delete:** Remove out-of-stock or discontinued coats from the system.

### 👤 User Mode
A highly interactive, widget-based shopping experience with the following features:
* **Browse & Filter:** Cycle through available coats one by one using "Next Coat", or quickly "Filter by Size" to find the perfect fit.
* **Visual Shopping:** As you browse, the image of the current trench coat automatically pops up on your screen so you know exactly what you are buying!
* **Shopping Cart:** Easily "Add to Cart" and "View Cart" to keep track of your pending purchases.
* **Exportable Cart:** Choose your preferred format (like HTML) to save your shopping cart, and use the "Open Cart File" button to view your receipt externally.

## 🛠️ Technologies Used
* **Language:** C++
* **Framework:** Qt (Widgets Application)
